# UK Sales Performance Analysis

## Project Overview
This project analyses sales transaction data for a simulated UK-based company to evaluate revenue performance, regional concentration, product mix, and trends over time.

## Business Objectives
- Analyse overall sales performance
- Identify key revenue drivers
- Evaluate regional and salesperson performance
- Analyse monthly revenue trends
- Provide actionable business recommendations

## Tools Used
- Excel (PivotTables, PivotCharts, Dashboards)

## Dataset
- Simulated UK sales transaction data
- Time period: January 2023 â€“ June 2024
- Includes order-level revenue, product, region, and salesperson data

## Key Analysis Areas
- Total revenue, total orders, and average order value (AOV)
- Revenue by region, product, and category
- Salesperson performance
- Revenue trends over time
- Revenue concentration by order size

## Key Insights
- Revenue is concentrated in a small number of UK regions
- A limited number of products drive the majority of revenue
- Revenue shows month-to-month volatility
- Salespeople exhibit different selling patterns (high-value vs high-volume)

## Recommendations
- Reduce regional dependency through targeted growth initiatives
- Prioritise high-performing products
- Introduce strategies to stabilise revenue volatility
- Use multiple KPIs for fair salesperson performance evaluation

- Full analysis report (PDF)
- Excel sales performance dashboard
